# 10. Current Output

## 이 브랜치의 의미

현재까지 학습 과정을 거쳐 나온 통합 산출물 브랜치입니다. 앞선 단계들의 결과가 한곳에 모여 있으며, 이후 확장 방향을 잡는 기준점 역할을 합니다.

## 현재 포함된 내용

- Study 도메인 CRUD
- Validation
- BaseEntity
- ApiResponse
- 전역 예외 처리
- DTO `record` 리팩토링
- 회원가입, 로그인
- MySQL 연결 설정
- Todo 도메인
- 사용자 정보 조회 기능
- 테스트 코드 일부

## 이 브랜치에서 보면 좋은 것

- 각 단계에서 추가된 기능이 최종 구조에서 어떻게 연결되는지
- 초기에 단순했던 코드가 공통화와 리팩토링을 거치며 어떻게 변했는지
- 도메인, 예외, 보안, 테스트가 한 프로젝트 안에서 만나는 방식

## 추천 복습 방법

1. `1-project-init` 부터 순서대로 브랜치를 이동하며 `STUDY.md` 를 먼저 읽습니다.
2. 그 다음 실제 코드 차이를 확인합니다.
3. 마지막에 이 브랜치로 돌아와 전체 구조를 다시 봅니다.
